# BATTI© - Premium Heels & Activewear

A modern e-commerce storefront for BATTI© built with React, Vite, TypeScript, and Shopify Storefront API.

## Tech Stack

- **React 18** + **TypeScript**
- **Vite** (build tool)
- **Tailwind CSS** (styling)
- **Shopify Storefront API** (product data & checkout)
- **Supabase** (auth & backend)
- **Vercel** (hosting)

## Getting Started

```bash
npm install
npm run dev
```

## Deployment

Deployed on Vercel. Push to `main` to trigger automatic deploys.
